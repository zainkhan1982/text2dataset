API Endpoints

GET /api/datasets

Get all community datasets

api_key (query) - Your API key

Example request:

curl "http://localhost:8006/api/datasets?api_key=YOUR_API_KEY"
GET /api/dataset/{dataset_id}

Get a specific dataset by ID

dataset_id (path) - ID of the dataset
api_key (query) - Your API key

Example request:

curl "http://localhost:8006/api/dataset/12345?api_key=YOUR_API_KEY"
GET /api/download/{dataset_id}

Download a dataset file

dataset_id (path) - ID of the dataset
api_key (query) - Your API key

Example request:

curl "http://localhost:8006/api/download/12345?api_key=YOUR_API_KEY" -o dataset.csv

Rate Limiting

The API is rate-limited to prevent abuse:

  • 100 requests per hour per API key
  • Downloads are not rate-limited

Usage Analytics

API usage is tracked for the following metrics:

  • Number of requests per endpoint
  • Request frequency
  • Data transfer volume
